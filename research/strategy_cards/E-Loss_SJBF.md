# E-Loss + SJBF 回填预测框架策略卡

* 
**策略名称**：EASY-Loss Machine Learning Prediction & Backfilling (E-Loss + SJBF) 


* **来源**：Gaussier, E., Reis, V., Glesser, D., & Trystram, D. (2015). Improving backfilling by using machine learning to predict running times. *SC '15*. 


* **原问题**：
* 在 HPC 批处理调度中，由于系统会杀掉超时的任务，用户通常会大幅度高估任务的请求运行时间 。


* 当前主流的调度器（如使用 EASY Backfilling 策略）假设用户提供的运行时间是准确的，严重的高估会导致回填调度策略失效，造成严重的系统资源浪费和较差的调度性能 。




* **核心机制**：
* 
**多项式回归预测**：利用在线的 $l_{2}$ 正则化二阶多项式模型对任务运行时间进行机器学习预测 。特征包含任务需求、用户历史提交数据和当前系统负载等 。


* 
**不对称损失函数（E-Loss）**：模型认为“低估”运行时间会破坏已排布的调度计划并导致任务饥饿，而“高估”仅导致资源短暂闲置 。因此，算法对高估使用平方损失，对低估使用线性损失 。


* 
**任务面积加权**：E-Loss 损失函数会根据任务的“面积”（$q_{j} \times p_{j}$，即资源需求乘以时间）进行加权惩罚 。


* 
**最佳回填组合**：使用预测时间配合 EASY-SJBF（最短任务优先回填）策略 ，并加上 INCREMENTAL（固定增量）修正机制（预测超时则按固定步长增加预估时间）。该组合使平均有界减速（$AVEbsld$）对比传统 EASY 降低了 28% 。




* **与本题的差异**：
* 原论文解决的核心痛点是“真实运行时间未知”，是在线调度 。本题是拥有全局视野的静态离线调度，每个任务的运行时长在提交时即绝对准确已知。


* 本题目标为 $E_{wait}$、$E_{finish}$ 和 $E_{memory}$ 三维指标，而原问题主要优化目标是平均有界减速（$AVEbsld$）。


* 原问题不涉及 GPU、CPU、内存、显存四维异构并发约束。


* **本题可采用部分**：
* **SJBF 回填思想（Shortest Job Backfilled First）**：原论文证明了优先用“短任务”去填补调度空隙能显著提升效率。在本题中，贪心预排布长任务/高优先级任务后，利用确定的极短任务见缝插针（Backfilling）将非常有价值。
* **基于任务面积的非对称特征权重**：原论文利用“任务面积”或资源需求给调度和惩罚进行加权的思想，可直接移植到本题 V1 阶段的多规则任务排序（Task Sorting）以及 V3 阶段的 LNS 破坏算子中。


* **计划修改**：
* **舍弃机器学习预测**：由于本题时长精确已知，无需实现在线预测与 INCREMENTAL 修正。直接利用真实时长进行准确无误的回填操作。
* **引入基于 SJBF 的 Deterministic Backfilling（确定性回填）**：在 V1 阶段贪心框架中，当一台服务器出现“高优任务暂无法启动而留下的时间空窗”时，按 SJBF（时长最短、所需 GPU 较少优先）策略扫描等待队列，将不会推迟原保留位置的碎片任务塞入。
* **任务排序公式引入“面积”乘子**：在设计贪心优先级评分时，将 $p \times GPU\_count$ 或 $\log(p \times GPU\_count)$ 作为一项可配置的惩罚/激励项。


* **预计改善指标**：
* $E_{wait}$（加权等待时间）：短任务优先回填能以极低成本让大量短任务提前完成，从而改善整体等待指标。
* $E_{memory}$（显存空闲）：有效利用由于长时任务资源未对齐而产生的显存时间窗碎片。


* **预计适用实例**：
* 任务时长方差大、存在明显长短任务分化、资源碎片密集的困难（Hard）与极端（Extreme）实例。


* **实现成本**：低-中。不需要额外的机器学习库，仅需在成员 A 的贪心状态查询逻辑中增加一个 Backfill 的扫描机制与二次排序即可。
* **实现者**：成员 A（负责 Backfilling 初版与任务排序特征）。
* **实验对照**：
* V1 Baseline：普通多规则贪心（无回填）。
* 对比 1：引入常规顺序回填的贪心。
* 对比 2：引入 SJBF（短任务优先）回填的贪心。
* 对比 3：在任务排序中测试 $\log(任务面积)$ 的权重消融实验。
